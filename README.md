# Bovis Visuals

보비스가 생성한 인터랙티브 HTML 시각화 결과물을 공개하는 GitHub Pages 저장소입니다.

## 공개 주소

- https://bohe1977.github.io/bovis-visuals/

## 구조

- `index.html` — 전체 시각화 목록
- 각 시각화는 날짜와 주제를 조합한 별도 폴더에 저장
  - 예: `2026-07-18-quantum-stocks/index.html`

## 게시 원칙

1. 결과물은 브라우저에서 독립 실행 가능한 HTML로 제작합니다.
2. 새 결과물은 고유한 하위 경로에 추가합니다.
3. 메인 페이지에서 제목·설명·생성일과 함께 링크합니다.
4. 개인정보, API 키, 토큰 및 비공개 데이터는 게시하지 않습니다.
